# Estrategia SEO para Cumplia

## Resumen Ejecutivo

Cumplia es una plataforma SaaS B2B que ayuda a empresas a cumplir con el Reglamento de IA de la UE (AI Act). Esta estrategia SEO se enfoca en captar tráfico cualificado de empresas europeas que necesitan soluciones de cumplimiento normativo para sus sistemas de IA.

## 7. Notas Técnicas

### Verificación Pendiente

Reemplazar en `app/layout.tsx`:
- `YOUR_GOOGLE_SEARCH_CONSOLE_CODE` → Código real de GSC
- `https://cumplia.com` → Dominio de producción
- `/og-image.jpg` → Imagen Open Graph real (1200x630)
- `/logo.png` → Logo para Schema.org

### Imágenes Requeridas

1. **OG Image** (1200x630px)
   - Branding Cumplia
   - Texto: "Cumple el AI Act con Confianza"
   - Colores corporativos

2. **Favicon**
   - favicon.ico (16x16, 32x32)
   - apple-touch-icon.png (180x180)

3. **Logo Schema**
   - Formato PNG con fondo transparente
   - Mínimo 112x112px

### Core Web Vitals Objetivo

- **LCP** (Largest Contentful Paint): < 2.5s
- **FID** (First Input Delay): < 100ms
- **CLS** (Cumulative Layout Shift): < 0.1
- **INP** (Interaction to Next Paint): < 200ms

## 9. Conclusión

La estrategia SEO de Cumplia se centra en:

1. **Captar intención de búsqueda alta** → "AI Act", "cumplimiento IA"
2. **Educar al mercado** → Contenido sobre regulación y obligaciones
3. **Convertir a usuarios** → Landing optimizada + flujo de registro claro

El mercado del AI Act es emergente con alta demanda de información. Cumplia tiene la oportunidad de posicionarse como la referencia en español para cumplimiento del AI Act.

**Prioridad inmediata:**
1. Verificar Google Search Console
2. Publicar guía completa del AI Act
3. Iniciar blog con contenido educativo
