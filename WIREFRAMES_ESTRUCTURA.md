## 🎬 Interactive Elements - Behavior Specs

### Feature Cards Animation (On Scroll)

```
Trigger: Card enters viewport (top 30%)

Timeline:
├─ 0ms:    opacity: 0, transform: translateY(80px)
├─ 300ms:  opacity: 1, transform: translateY(0px)
│          cubic-bezier(0.34, 1.56, 0.64, 1)
│
└─ On Hover:
   ├─ transform: translateY(-4px)
   └─ box-shadow: 0 8px 32px rgba(255,255,255,0.1)
```

### Button Interaction States

```
Button States:
├─ Default
│  ├─ background: #5c5cff
│  ├─ color: #ffffff
│  └─ shadow: 0 4px 12px rgba(92, 92, 255, 0.3)
│
├─ Hover
│  ├─ background: #7575ff (lightened)
│  ├─ transform: translateY(-2px)
│  └─ shadow: 0 8px 24px rgba(92, 92, 255, 0.5)
│
├─ Active
│  ├─ transform: translateY(0px)
│  └─ shadow: 0 2px 8px rgba(92, 92, 255, 0.3)
│
└─ Focus (Keyboard)
   └─ outline: 2px solid #ffffff (WCAG AA)
```

### Testimonial Carousel

```
Navigation:
├─ Dots: 3 visible indicators
├─ Auto-advance: 6s per slide
├─ Manual: Click dots or arrows
│
Transition:
├─ Duration: 500ms
├─ Effect: Fade + Slide
│  ├─ Outgoing: opacity: 0, translateX(-20px)
│  └─ Incoming: opacity: 1, translateX(0px)
│
Accessibility:
├─ ARIA labels for carousel
├─ Keyboard: Left/Right arrows
└─ Pause on hover/focus
```

### Mouse Tracking (Hero Graphic)

```javascript
// Pseudo-code for mouse tracking effect

addEventListener('mousemove', (e) => {
  const x = (e.clientX / width) - 0.5;  // -0.5 to 0.5
  const y = (e.clientY / height) - 0.5; // -0.5 to 0.5

  gsap.to('.hero-graphic', {
    x: x * 100,              // Max 50px movement
    y: y * 50,               // Max 25px movement
    rotationY: x * 10,       // Max 5° rotation
    rotationX: -y * 10,      // Max -5° rotation
    duration: 0.5,
    ease: 'power2.out',
  });
});

// Fallback for prefers-reduced-motion
@media (prefers-reduced-motion: reduce) {
  .hero-graphic { transform: none !important; }
}
```

## 📊 Performance Targets

```
Metric               Target      Implementation
─────────────────────────────────────────────────────
Largest Contentful Paint (LCP)
                    < 2.5s      Hero image/video optimized
                               Critical CSS inlined

First Input Delay (FID)
                    < 100ms     Minimal JavaScript
                               GSAP for animations

Cumulative Layout Shift (CLS)
                    < 0.1       Fixed dimensions
                               No surprise layout changes

First Contentful Paint (FCP)
                    < 1.8s      Preload critical fonts
                               SVG icons inline

Time to Interactive (TTI)
                    < 3.8s      Lazy load below-fold content
                               Code-split animations
```
